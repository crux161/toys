Introduction

The following is a list of 10 programs everyone should understand the concepts of and how to implement them in actual code (hopefully across several languages, but initially with a focus on rust).

Hello World:

A simple program that prints the message "Hello, World!" to the console. This program is often used as an introduction to programming and is a good starting point for learning a new programming language.

Calculator:

A program that performs simple arithmetic calculations such as addition, subtraction, multiplication, and division. This program can be extended to perform more complex calculations such as trigonometric functions, logarithms, and exponentials.

FizzBuzz:

A program that prints the numbers from 1 to 100, but for multiples of three, it prints "Fizz" instead of the number, and for multiples of five, it prints "Buzz". For numbers that are multiples of both three and five, it prints "FizzBuzz". This program tests a programmer's ability to use conditional statements and loops.

Factorial:

A program that computes the factorial of a given number. The factorial of a number is the product of all the integers from 1 to that number. For example, the factorial of 5 is 5 * 4 * 3 * 2 * 1 = 120. This program tests a programmer's ability to use recursion.

Binary Search:

A program that implements the binary search algorithm to find the position of a given value in a sorted list of values. This program tests a programmer's ability to use arrays and implement algorithms.

Sorting:

A program that implements a sorting algorithm such as bubble sort, selection sort, insertion sort, or merge sort. This program tests a programmer's ability to implement algorithms and use arrays.

File Input/Output:

A program that reads data from a file and writes data to a file. This program tests a programmer's ability to work with files and use file input/output functions.

Tic Tac Toe:

A program that implements the game of Tic Tac Toe. This program tests a programmer's ability to use conditional statements, loops, and implement game logic.

Palindrome:

A program that checks whether a given string is a palindrome or not. A palindrome is a word, phrase, number, or other sequence of characters that reads the same forward and backward. This program tests a programmer's ability to use loops and string manipulation functions.

Random Number Generator:

A program that generates random numbers within a given range. This program tests a programmer's ability to use random number functions and implement basic algorithms.
